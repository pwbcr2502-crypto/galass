## Quick Start

### Backend
```bash
cd backend
npm install
npm run dev
```

### Mobile Frontend
```bash
cd frontend/mobile
npm install
npm run dev
```

### Big Screen Display
```bash
cd frontend/bigscreen
npm install
npm run dev
```

## Tech Stack

- **Backend**: Node.js + Express + PostgreSQL + Redis + WebSocket
- **Mobile**: Vue3 + Vant UI + Axios
- **Big Screen**: Vue3 + ECharts + WebSocket
- **Database**: PostgreSQL + Redis
